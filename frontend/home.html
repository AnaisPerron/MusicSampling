About Sample Tree

Project Overview

Sampling is a fundamental technique in modern music production, where artists reuse portions of existing songs to create new compositions. Our project, Sampling Tree, is an interactive data visualization website that allows users to explore how songs are interconnected through sampling. By leveraging open data from WhoSampled, we build dynamic tree structures that map out the relationships between original tracks and their sampled counterparts, offering a unique and insightful way to discover music lineage.

Objectives

Visualizing Song Lineage: Our platform creates an interactive tree structure where users can see how a song has influenced other tracks through sampling. Leveraging Open Data: We use WhoSampled's publicly available data to fetch and analyze song-sample relationships. Data Structure Implementation: We utilize tree structures to model and display the hierarchy of sampled songs effectively. User Interaction and Exploration: Users can search for songs, expand nodes to reveal more connections, and explore entire musical lineages interactively.
